# Getting Started with 2Bot

Welcome to 2Bot — a modular automation platform that lets you connect services, install plugins, and build workflows.

## Quick Start Steps

1. **Create your account** — Sign up at dash.2bot.org
2. **Set up a Gateway** — Connect your first service (e.g., Telegram Bot, AI provider)
3. **Install a Plugin** — Browse the Plugin Store and install automation modules
4. **Create a Workflow** — Chain plugins together with triggers

## Key Concepts

### Gateways
Gateways are connections to external services. Currently supported:
- **Telegram Bot** — Connect your Telegram bot using a bot token
- **AI Providers** — Connect OpenAI, Anthropic, or other AI services with your API keys
- **Webhooks** — Receive and send data via HTTP webhooks

### Plugins
Plugins are automation modules you install from the marketplace. Each plugin has:
- Required gateway types (what services it needs)
- Configuration options
- Input/output schemas for workflow integration

### Workflows
Workflows let you chain plugins together:
- **Triggers**: Telegram message, scheduled time, webhook, or manual
- **Steps**: Process data through plugins in sequence
- **Scopes**: Personal, department, or organization-wide

### Credits
Credits are the universal currency for using 2Bot's built-in AI features. Your plan determines your monthly credit allocation.
