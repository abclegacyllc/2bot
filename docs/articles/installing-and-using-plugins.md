# Installing and Using Plugins

Plugins are the building blocks of automation on 2Bot. Here's how to find, install, and use them.

## Finding Plugins
1. Go to **Dashboard → Plugin Store**
2. Browse by category: General, Analytics, Messaging, Automation
3. Use the search bar to find specific functionality

## Installing a Plugin
1. Click on a plugin to see its details
2. Review the **required gateways** — you need these set up first
3. Click **"Install"**
4. Configure the plugin settings if needed

## Plugin Configuration
Each plugin has its own configuration:
- Some plugins work out of the box
- Others need gateway connections (e.g., which Telegram bot to use)
- Advanced plugins have detailed config options

## Managing Your Plugins
Go to **Dashboard → My Plugins** to:
- View all installed plugins
- Configure plugin settings
- Enable/disable plugins
- Uninstall plugins you no longer need

## Using Plugins in Workflows
Plugins become powerful when combined in workflows:
1. Create a new workflow
2. Set a trigger (e.g., "When Telegram message received")
3. Add plugin steps in sequence
4. Map data between steps

## Common Issues
- **"Missing gateway"** — Install the required gateway type first
- **"Plugin not responding"** — Check if the connected gateway is functional
- **"Execution failed"** — Check your credit balance and plugin configuration
